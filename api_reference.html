APIリファレンス

このセクションでは、py2hackCraftの主要なクラスとメソッドについて説明します。

基本クラス

Player

サーバーとの接続を管理するクラスです。

Entity

エンティティの操作を管理するクラスです。

Volume

3次元の領域を表すクラスです。

LocationFactory

座標を生成するファクトリクラスです。

データクラス

Location

座標を表すデータクラスです。

Block

ブロックの情報を表すクラスです。

ItemStack

アイテムスタックの情報を表すクラスです。

イベントクラス

InteractEvent

エンティティとの相互作用イベントを表すクラスです。

EventMessage

イベントメッセージを表すクラスです。

ChatMessage

チャットメッセージを表すクラスです。

RedstonePower

レッドストーンの出力レベルを表すクラスです。
