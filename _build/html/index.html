class py2hackCraft.modules.Coordinates

Minecraftの座標系を表すクラス

座標系の種類: - ABSOLUTE: 絶対座標 (例: 100, 64, -200) - RELATIVE: 相対座標 (例: ~10, ~0, ~-5) - LOCAL: ローカル座標 (例: ^0, ^5, ^0)

static absolute(x: int, y: int, z: int) → tuple

絶対座標を指定する

パラメータ:

  • x (int) -- X座標

  • y (int) -- Y座標

  • z (int) -- Z座標

戻り値:

(x, y, z, 座標系)

戻り値の型:

tuple

サンプル

>>> Coordinates.absolute(100, 64, -200)
(100, 64, -200, "")
static local(x: int, y: int, z: int) → tuple

ローカル座標を指定する（自分の向きを基準とした前後左右）

パラメータ:

  • x (int) -- 右(+)左(-)方向の相対距離

  • y (int) -- 上(+)下(-)方向の相対距離

  • z (int) -- 前(+)後(-)方向の相対距離

戻り値:

(x, y, z, 座標系)

戻り値の型:

tuple

サンプル

>>> Coordinates.local(0, 5, 0)  # 自分の真上5ブロック
(0, 5, 0, "^")
static relative(x: int, y: int, z: int) → tuple

相対座標を指定する（自分を中心とした東西南北）

パラメータ:

  • y (int) -- 上(+)下(-)方向の相対距離

  • z (int) -- 南(+)北(-)方向の相対距離

戻り値:

(x, y, z, 座標系)

戻り値の型:

tuple

サンプル

>>> Coordinates.relative(10, 0, -5)  # 東に10、北に5進む
(10, 0, -5, "~")
